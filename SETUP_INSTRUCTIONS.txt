🚀 QUICK SETUP (60 seconds)

1. Start the server:
   
   npm run dev

2. Open browser:
   
   http://localhost:3000

3. ⚠️  ENABLE MOBILE VIEW:
   
   Chrome/Edge:
   • Press F12 or Ctrl+Shift+I (Cmd+Option+I on Mac)
   • Press Ctrl+Shift+M (Cmd+Shift+M on Mac)
   • Select "iPhone 14 Pro" from dropdown
   • Refresh page (F5)

4. ✅ Verify you see:
   
   ✓ Yellow "Get the Valeo App" bar at top
   ✓ Bottom navigation bar with 4 icons
   ✓ Mobile-optimized layout
