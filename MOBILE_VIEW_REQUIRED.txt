  📱 QUICK SETUP (Choose Your Browser)

  ┌────────────────────────────────────────────────────────────┐
  │  CHROME / EDGE (Recommended)                               │
  ├────────────────────────────────────────────────────────────┤
  │  1. Press F12 (or Ctrl+Shift+I / Cmd+Option+I)            │
  │  2. Press Ctrl+Shift+M (or Cmd+Shift+M on Mac)            │
  │  3. Select "iPhone 14 Pro" from dropdown                   │
  │  4. Press F5 to refresh                                    │
  └────────────────────────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────┐
  │  FIREFOX                                                    │
  ├────────────────────────────────────────────────────────────┤
  │  1. Press F12 (or Ctrl+Shift+I / Cmd+Option+I)            │
  │  2. Press Ctrl+Shift+M (or Cmd+Option+M on Mac)            │
  │  3. Select a mobile device from presets                    │
  │  4. Press F5 to refresh                                    │
  └────────────────────────────────────────────────────────────┘

  ┌────────────────────────────────────────────────────────────┐
  │  SAFARI (Mac)                                               │
  ├────────────────────────────────────────────────────────────┤
  │  1. Enable Develop menu (Preferences > Advanced)           │
  │  2. Press Cmd+Option+I                                     │
  │  3. Develop > Enter Responsive Design Mode                 │
  │  4. Select "iPhone 14 Pro"                                 │
  │  5. Press Cmd+R to refresh                                 │
  └────────────────────────────────────────────────────────────┘


  ✅ VERIFY YOUR SETUP

  After enabling mobile view, you should see:

    ✓ Yellow "Get the Valeo Health App" bar at the top
    ✓ Bottom navigation bar with 4 icons (Home, Referrals, Plans, Wallet)
    ✓ Mobile-optimized layout (single column on mobile)
    ✓ Proper spacing and touch-friendly buttons


  ❌ WITHOUT MOBILE VIEW YOU WILL MISS:

    • App redirect bar (only shows <768px width)
    • Proper bottom navigation layout
    • Mobile-optimized card layouts
    • Touch-optimized button sizes
    • The intended mobile-first experience
    • Key features that only work in mobile mode


  📐 RECOMMENDED DIMENSIONS

    iPhone 14 Pro:  390 x 844 pixels  ⭐ BEST CHOICE
    Google Pixel 7: 412 x 915 pixels
    iPhone SE:      375 x 667 pixels  (minimum)


  🔧 TROUBLESHOOTING

    Problem: "I don't see the yellow bar at top"
    Solution: Ensure width is < 768px, refresh the page

    Problem: "Bottom nav looks stretched"
    Solution: Select a mobile device preset, not "Responsive"

    Problem: "Can't find device toggle button"
    Solution: Look for phone/tablet icon in DevTools toolbar
